## Overview
Simple blockchain wrapped with a small express API.

## Getting Started

- `npm i`
- `node app.js`

## Endpoints

- Get /block/:blockheight
    - returns the block at that height or an error message

- Post /block
    - returns the block placed in the DB or an error
